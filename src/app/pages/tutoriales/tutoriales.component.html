Ejercicio 13:

Para saber las versiones que tienes instaladas de TypeScript, Node.js, el instalador de paquetes y Angular, podemos utilizar los siguientes comandos en la consola:

  • Para TS: tsc -v
  • Para Node: node -v
  • Para instalador de paquetes: npm -v
  • Para Angular: ng version
