Ejercicio 12 y 13

Ejercicio 12:



Para crear un archivo TypeSript, simplemente creamos un archivo en nuestro editor con la extensión .ts --> index.ts .Podemos darle un contenido como un console.log... A continuación debemos asegurarnos de que tenemos typescript con 'tsc -v' y si no lo tenemos instalarlo con 'npm install -g typescript'. A continuación transpilariamos el archivo desde la consola con el comando 'tsc index.ts' y nos creara un archivo llamado 'index.js' el cual seria la transpilación de index.ts.


Ejercicio 13:



Para saber las versiones que tienes instaladas de TypeScript, Node.js, el instalador de paquetes y Angular, podemos utilizar los siguientes comandos en la consola:
- Para TS: tsc -v
- Para Node: node -v
- Para instalador de paquetes: npm -v
- Para Angular: ng version
